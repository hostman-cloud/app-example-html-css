# HTML & CSS

Example of a static [HTML & CSS](https://developer.mozilla.org/en-US/docs/Web/HTML) application that can be deployed to **Hostman** without configuration.

:tada: [Demo](https://hostman-cloud-app-example-html-css-3e85.hostman.dev/)

:rocket: [Create your Apps](https://hostman.com/my/apps/create)

:books: [Hostman Apps Documentation](https://hostman.com/docs/app-platform/backend-apps/)

## <a name="dev"></a>Running locally

This is a static application — no build step is required. Open `index.html` directly in your browser, or serve the folder with any static server:

```bash
# serve the current folder on http://localhost:3000
npx serve .
```
